## Installation Guide

Please have
![](https://img.shields.io/badge/Node-16.14.0-success?style=plastic&logo=nodedotjs)

1. Clone Repository

```shell
$ git clone <repository>
```

2. Install Dependencies

```shell
$ yarn install
```

3. Create .env.local file
   -No env needed yet-

## Run development server

```shell
$ yarn start
```
